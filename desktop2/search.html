Last Updated: {{ site.time | date:"%Y-%m-%d" }}

Search List Overview

The Search function can be activated by clicking the Search Button on the Top Bar. When clicked the search line expands allowing the user to type in what they are searching for. As they type the search box below is populated with results. Users will be able to fine-tune search by using a built-in filter system, for example to search for Naruto but exclude the movies you would type in Naruto -Movie. Please note, the built-in filter system has not been finalized and is subject to change.

As an overlay, the Search List can be displayed over any section and is automatically closed when the user clicks outside the Pinned List area. Results are shown in a compressed format and expand on hover which is shown in action in the mockup on the right. If the users search returns more results then can be shown, the user can scroll through the list.
