Overview of the Shoko Desktop 2 Frame.

Last Updated: {{ site.time | date:"%Y-%m-%d" }}

Frame Overview

The Collapsed Frame, as seen in the image on the right is the default look for Shoko Desktop V2. The side-menu is in its collapsed state giving the Content Area more display space. Users can toggle the Extended Frame by clicking the toggle button located at the bottom of the side-menu. Please note this button is not currently shown in the mockups as the Settings button is shown at the bottom.

The minimum supported resolution for Shoko Desktop 2 is 1366X768, users with higher resolutions will be able to use the Extended Frame without losing display space for the Content Area.

Top Bar

The Top Bar is a fixed global bar that contains the following items.
Item Description
Queue Processor Count Visual indicator representing how many queued items there are to process. This number does include items currently being processed but since its shown as a single number, we assume they are remaining items and add them to the list.
Action Items Count Visual indicator representing how many issues need your attention. Action Items represent a task where user intervention is required, these include but are not limited to Unrecognized Items, Multiple Files and Duplicate Files.
Pinned Button When clicked, shows the pinned slide-out pinned list.
Search Button When clicked, shows the pinned slide-down search menu.
Discord Button When clicked, will open our Discord server in your browser.
About Button When clicked, will show the About window with program relevant information.
Minimize Button When clicked, will minimize the program.
Maximize Button When clicked, will maximize the program, clicking it again will return the window to its previous size.
Exit Program Button When clicked, will close the program.

Side-Menu

The Side-Menu is a fixed global menu that allows the user to switch to the various sections. Please note that while displayed, the User Avatar does nothing.

Item Description
Dashboard Takes the user to the Dashboard section.
Collection Takes the user to the Collection section.
Calender Takes the user to the Calender section.
My Lists Takes the user to the My Lists section.
Playlists Takes the user to the Playlists section.
Utilities Takes the user to the Utilities section.
Server Takes the user to the Server section.
Settings Shows the Settings overlay on top of the current section.
User Items User defined buttons that allow the user to create custom shortcuts. A maximum amount of 4 user items can be shown.
Menu Toggle When toggled, changed the frame type.
