Pinned List Overview

While similar to Shoko Lists, the Pinned list is designed as a quick look list for the users immediate need. When clicked, a pinned item will take the user directly to the series and will not be removed from the Pinned List until the users clicks the Unpin Button below the series image. Additionally, the pinned series can be added to a Shoko List by clicking the Add to Shoko List Button.

As an overlay, the Pinned List can be displayed over any section and is automatically closed when the user clicks outside the Pinned List area. If the user has more items pinned than can be displayed, the user will be able to scroll through the list. At this time it's undecided if there will be a limit to the amount of pinned series there can be. Users will also be able to pin series not in their collection.
