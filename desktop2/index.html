Information on Shoko Desktop 2, our new desktop program being built.

What is Shoko Desktop 2?

Shoko Desktop 2, also known as ClientV2 and DesktopV2 is the next version of Shoko Desktop we are creating. Unlike the current iteration of Desktop which is built with C# and uses WPF for the UI, we'll be using Electron which allows us to create a desktop program using web languages.With a complete overhaul of the UI and underlying code, we'll be able to overcome currently present limitations as well as make continued development and contributions from outside sources much easier.

Development Stages

We currently do not have an ETA for when Shoko Desktop 2 Version 1.00 will be out.

We've broken our development cycle into multiple stages to allow our users to better track our progress. Please note, these development stages are not equal so we cannot provide a time breakdown for each one.

  • Initial Plans & Mockups
  • Finalize Plans & Mockups
  • Initial Framework Coding
  • Section Coding
  • Section Finalizing
  • CSS Finalizing

Current Stage: Finalize Plans & Mockups

Looking To Help?

Shoko is a completely volunteer project maintained by a small group of developers with assistance from it’s community. As with most open-source and volunteer projects, time is the most precious commodity with all work being done for free and thus we’re always looking for users who are eager to help out.
