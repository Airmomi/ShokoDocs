Action Items Overview

Action Items are a combination of configurable items that require the users attention to resolve. While primarily designed to alert the user to import issues, plugin developers can create an action item type that will be tracked and alert the user when needed. Below is a list of the default action item types which can be enabled/disabled in the settings section.

Default Action Items

  • Unrecognized Files
  • Multiple Files
  • Duplicate Files
