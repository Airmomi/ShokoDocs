Information on Filters Shoko Desktop 2.

Last Updated: {{ site.time | date:"%Y-%m-%d" }}

Filter List Overview

Filters function in the same manner as they did in Shoko Desktop, the big difference is their design. In Shoko Desktop 2, each filter is shown in a scalable Poster Grid format that utilizes LazyLoad to reduce additional wait time being added when switching filters. The Series Name is shown at the bottom in an overlay on the poster, for series with exceptionally long titles, ellipses are used once the max length has been reached. While not pictured, there will be visual indicators for Unwatched Count and Groups shown on the series poster.

The Filter Menu is not shown by default, instead the last viewed filter is loaded by default when clicking the Collectionsbutton.. Switching filters is done by clicking the Collections icon when its the active icon, to access the Filter Menu while not in the Collection section, the Collections icon needs to be Double-Clicked. Adding a new filter is done by clicking the Blue button at the top of the Filter Menu, this will display a new window. Editing an existing filter is done by clicking on the Settings icon in the Filter Panel Header.

Filter - Series Preview

After hovering over a series for two seconds, the Series Preview overlay is displayed providing Series and Collection information related to that series. Like the Series Poster itself, the Series Fanart will randomly change when the Series Preview is accessed. Below the Series Description are the User buttons that allow the user to either Pin The Series or add it to a Shoko List. At the bottom are the series tags, custom tags are shown first followed by series tags.

Adding & Editing Filters

The Adding & Editing Filters panel is broken into three parts; Filter Settings, Conditions and Filter Results. Each section is designed to be simple and self-explanatory as managing filters should not be considered difficult. Existing Filter Conditions can be edited or removed by clicking the corresponding icon at the right of the Condition. Adding a new Condition is done by clicking the Add button at the bottom, while users select the first part, the Condition itself, the second part is pre-populated based on the user's choice wih the last part being a user input based on the second part. This should reduce confusion on how to properly set up filters as Shoko will set them up, users just need to input their preference.
