Shoko Desktop - Utilities Tab

The Utilities allows you to finetune Shoko to manage your collection.

Unrecognized

In the Unrecognized tab you will find all the files that shoko was not able to recognize automatically. This could be due to multiple reasons:

  • Unknown release
  • Failed Hash
  • Not a video file
A few functions are available to you to manage all the files in Bulk:
Available Actions Description
Refresh The Refresh button will refresh your view, and add any files that might not have been cached to the Desktop Client.
Rescan All Forces ShokoServer to rescan AniDB for your unrecognized files with the current Hash.
Rehash All Forces ShokoServer to rehash the files, this will be usefull when a hash runs into issues.
Search / Logs Self-explanatory, search the list, or view the log.

When you select an unrecognized file you will receive a few more options besides the Rescan and Rehash as described above (but these will only run for the selected files).

Next to rescanning or rehashing the files, you can also manually link a file to an existing (or new) series, this will be usefull when there is no AniDB Hash yet.

To manually link a file to an episode or movie, after selecting the file, select the series you want the file to be connected with, choose the episode number (or range of episodes, Shoko will try to match the number of the episodes in bulk this way), and press Confirm at the top.

To add a new series press the "New Series" and use the Search window to add the series to your Collection. Then link the file as you would with an existing Series.

Alternatively press the Ignore button to ignore the file so that Shoko will not try to match the file in the future.

Ignored

This displays the list of Ignored files. You can do a few things with these files.

Available Actions Description
Play Play the file
Delete Delete the file from the system
Restore Restore the file to the Unrecognized tab
Open Folder Open the folder where the file is located

Manually Linked

The list of all files that were manually linked to their series / episode.

The "Rescan All" button will force ShokoServer to scan AniDB if a release is available for all of the files in the list. When opening the file info by pressing the + you will receive some more options for the specific file.
Available Actions Description
Play Play the file
Rescan Force a scan of AniDB
Delete Link Remove the link to the series/episode and place the file back into the Unrecognized List
